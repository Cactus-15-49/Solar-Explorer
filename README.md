# Solar Explorer

## Installation

### Requirements

-   [Composer](https://getcomposer.org)
-   [Valet](https://laravel.com/docs/8.x/valet) or [Homestead](https://laravel.com/docs/8.x/homestead)
-   `CORE_WALLET_SYNC_ENABLED=true` on the node .env
